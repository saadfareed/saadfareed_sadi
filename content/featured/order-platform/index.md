Architected a polyglot persistence layer for high-throughput order flows: MongoDB for menus and social content, Cassandra for high-write order events, and Elasticsearch for sub-<strong>100ms</strong> search. Built a real-time notification backbone with MQTT and Redis pub/sub for stage-specific customer and store updates.
