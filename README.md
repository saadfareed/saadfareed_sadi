# saadfareed_sadi

A personal website repository for Saad Fareed built with Gatsby.

This project showcases personal projects, featured work, and technical notes focused on AI, cybersecurity, cloud computing, and software development.

## 🚀 Project Overview

- Gatsby-powered static site
- Markdown content under `content/` for featured posts, jobs, and projects
- React + styled-components for UI
- SEO, sitemap, offline support, and analytics configured via Gatsby plugins

## 📁 Repository Structure

- `src/` — React components, pages, styles, and utilities
- `content/` — Markdown content for projects, featured articles, and jobs
- `static/` — Static assets served directly by Gatsby
- `gatsby-config.js` — Site metadata and plugin configuration
- `package.json` — Dependencies and scripts

## 💻 Development

Install dependencies and run the site locally:

```bash
npm install
npm run develop
```

Then open `http://localhost:8000` in your browser.

## 🛠️ Build

Create a production build with:

```bash
npm run build
```

Serve the built site locally:

```bash
npm run serve
```

## ✨ Maintenance

Format files with Prettier:

```bash
npm run format
```

Clean Gatsby cache:

```bash
npm run clean
```
